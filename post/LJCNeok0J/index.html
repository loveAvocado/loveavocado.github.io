· 08-03-2022 ·

在实践中总结 HTML+CSS 用法

Basic Knowledge of CSS

css selector format 样式选择器格式

元素样式

element {
    property: value;
}

类样式

.class {
    property: value;
}

margin 边缘 意义

Think of the margin as invisible space around an element. Using these two margin properties, center the div element within the body element.
边缘就是元素周围“隐形”的空间。

.flag-top {
    height: 4px;
    background-color: #0066cc;
    border-color: #0066cc;
    margin-bottom: 0px;
}

.flag-bottom {
	height: 4px;
	background-color: #ffcc00;
	border-color: #ffcc00;
	margin-top: 0px;
}

padding

You can give your menu some space between the content and the sides with various padding properties.
在内容和块边缘之间填充一些空间; 可以是相对向外扩张，也可以是向内卷曲。

elements {
	padding-left: 20px;
	padding-right: 20px;
	padding-bottom: 20px;
	padding-top: 20px;
	padding: 20px;
}

Basic Knowledge of HTML5 Elements

p 段落

<p> elements are block-level elements, so they take up the entire width of their parent element. p 元素是块级别的元素，因此会占用整行其祖元素的宽度。

  • To make 2 p elements behave more like in the same line, you need to apply some styling to the p elements. （如，将width属性调整为50%，并将两个p元素写在同一行。）

    <article class="item">
	<p class="flavor">French Vanilla</p><p class="price">3.00</p>
</article>

div 区块划分

何时使用 <div> —— 设计排版

The div element is used mainly for design layout purposes.

用于设计排版和布局。

width 宽度表示

  1. 百分比（%）
    Change the width property's value to be 80%, to make it 80% the width of its parent element (body).
    即其祖元素的宽度的百分比。
    如果仅为一行长度较长的字符串提供较短（如25%）的宽度空间，some of the text would start wrapping around to the next line.

  2. 像素值（px）

article 相关的数据

article elements commonly contain multiple elements that have related information.

article 元素通常用于包含相互关联的诸多元素，如 Cafe 练习中的价位表。

a 链接

The default color of a link that has not yet been clicked on is typically blue. The default color of a link that has already been visited from a page is typically purple. 未被点击的链接的默认颜色是蓝色，而被访问过的则是紫色。

css selectors of a

可以使用a选择器中的color改变其默认颜色。

可以使用虚拟选择器a:visited改变连接被访问后的颜色；（You change properties of a link when the link has actually been visited by using a [[pseudo-selector]] that looks like a:visited { propertyName: propertyValue; }.）

a:visited {
		color: grey;
}

可以使用虚拟选择器a:hover设置鼠标悬浮于链接之后的样式

a:hover {
	color: white;
}

使用 a:active 可以设置链接被点击时的样式。

hr 空心分割线（块）

horizontal line 分割线
self closing

height of hr

通过调整 hr 的 height 属性，可以将其修改为高度为 x px 的空心矩形。

background-color of hr

用于填充 hr 的颜色。

border-color of hr

hr 边框的颜色，边框的宽度为1px；

img 图片

img {
	display: block; /*将图片视为block以修改其样式*/
	margin-left: auto;
	margin-right: auto;
}
