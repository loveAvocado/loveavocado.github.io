功能指南

字幕制作类

旧版标题工具

新建字体：新建 -> 旧版标题 -> 点击左上角的文字工具，画一个稍大的框，打入文字 -> 全选所有文字后可以在右侧选择填充颜色 -> 使用左侧圆角矩形等工具可制作字体框 -> 修改矩形框长度时可按住option键 ->
路径文字工具：可以使字体沿着所画的曲线排列；
竖版横版文字工具：沿着纵向或横向排列文字；
描边效果： 右侧工具栏可描边效果；

关键帧动画

选中目标素材-> 进入效果控件 -> 进入运动属性 -> 点亮「位置」属性前面的按钮（关键帧按钮） -> 下图所示⭕️处出现一个关键帧-> 拖动时间轴并移动目标素材位置可创建新的关键帧（类似于Flash动画制作）

同样地，还可以在不同关键帧改变目标素材的其他属性以营造更多效果；
批量操作：粘贴属性command+option+V（粘贴时会弹出一个对话框，只需保留需要的属性即可），注意这里所粘贴的关键帧属性是等比例而非等时长粘贴的，因此或需要手动修改；

蒙版

作用：框定范围，仅显示所框定范围内的内容；
细节：将素材窗口下的「适合」调整为「25%」有助于框选素材；
钢笔工具：点选端点后可滑动杠杆使得所画线条弯曲，如需一次只转动一根杠杆，可按住option键；

第四课 手机拍摄

曝光三要素

内容 作用 特点及影响 参考设置
光圈 决定镜头打开的大小； 大光圈更能突出主体，虚化背景；手机一般无法调整光圈 需要背景虚化F2.8左右，不需要F5.0以上
快门 决定感光元件前幕帘的闭合快慢，即镜头的开启时间； 影响画面的亮度大小以及有无动态模糊； 快门一般设置为视频帧数的两倍，如：30fps的视频用1/60的快门；24fps的用近似1/50的快门；50倍数的快门速度在LED光源下拍照可以防止频闪；一般固定在1/50，除非要拍流光效果(慢速快门）。
ISO 数字越大，感光元件对光线越敏感；相应地，噪点也会越多； 保持正常曝光，ISO越低越好。ISO较低后期可以补救，ISO过高，后期很难补救； 一般1600以下即可，画面太难需要灯光辅助或调大光圈

备注

升格镜头：电影摄影中的一种技术手段，电影摄影拍摄标准是每秒24帧，也就是每秒拍摄24张，这样在放映时才能是正常速度的连续性画面，但为了实现一些简单的技巧，比如慢镜头效果，就要改变正常的拍摄速度，比如高于60帧/秒，这就是升格，放映效果就是慢动作。如果降低拍摄速度（低于24帧/秒），就是降格，放映效果就是快动作。
